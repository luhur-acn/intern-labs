# Phase 0: Environment Setup 🛠️

Langkah pertama dalam proyek ini adalah menyiapkan lingkungan kerja dengan alat yang diperlukan dan menginisialisasi backend state Terraform secara manual.

## 0.1 Instalasi Tools

Jalankan perintah berikut di terminal sandbox Anda untuk menginstal Terraform dan Terragrunt.

```bash
# Install Terraform v1.14.6 (Latest Stable)
curl -Lo /tmp/tf.zip https://releases.hashicorp.com/terraform/1.14.6/terraform_1.14.6_linux_amd64.zip
unzip -o /tmp/tf.zip -d /usr/local/bin/

# Install Terragrunt v0.99.4 (Latest Stable)
curl -Lo /usr/local/bin/terragrunt https://github.com/gruntwork-io/terragrunt/releases/download/v0.99.4/terragrunt_linux_amd64
chmod +x /usr/local/bin/terragrunt

# Verifikasi Instalasi
terraform version
terragrunt --version
```

## 0.2 Pembuatan State Backend (Manual)

Karena Terraform tidak dapat mengelola storage tempat ia menyimpan state-nya sendiri di awal, kita harus membuat S3 Bucket dan DynamoDB Table secara manual.

> **Catatan**: Ganti `[yourname]` dengan nama unik Anda.

### 1. Buat S3 Bucket untuk Terraform State

```bash
aws s3api create-bucket \
  --bucket iac-capstone-tfstate-[yourname] \
  --region us-east-1
```

### 2. Aktifkan Versioning pada Bucket

Versioning sangat penting agar kita bisa kembali ke versi state sebelumnya jika terjadi kerusakan.

```bash
aws s3api put-bucket-versioning \
  --bucket iac-capstone-tfstate-[yourname] \
  --versioning-configuration Status=Enabled
```

### 3. Aktifkan Enkripsi Sisi Server (SSE)

```bash
aws s3api put-bucket-encryption \
  --bucket iac-capstone-tfstate-[yourname] \
  --server-side-encryption-configuration \
    '{"Rules":[{"ApplyServerSideEncryptionByDefault":{"SSEAlgorithm":"AES256"}}]}'
```

### 4. Buat Tabel DynamoDB untuk State Locking

State locking mencegah dua orang melakukan deploy secara bersamaan yang dapat merusak file state.

```bash
aws dynamodb create-table \
  --table-name iac-capstone-tfstate-lock \
  --attribute-definitions AttributeName=LockID,AttributeType=S \
  --key-schema AttributeName=LockID,KeyType=HASH \
  --billing-mode PAY_PER_REQUEST
```
